# Dashboard Frontend

React + Vite frontend for the operations dashboard template.

## Development
```bash
npm install
npm run dev
```

The UI expects backend APIs at `http://localhost:8788` (proxied via Vite).

## Production Build
```bash
npm run build
npm run preview
```

## Active Surface
- Module health/status cards
- Operator actions for module lifecycle
- Readiness checks and runtime summary
- Event stream panel
- Component Gallery tab with reusable domain-neutral UI blocks
